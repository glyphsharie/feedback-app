# Feedback API Pairing Exam

## Overview
This is a minimal full-stack TypeScript app for collecting feedback.

## Setup

### Backend
```bash
cd server
npm install
npm run dev
```

### Frontend
```bash
cd client
npm install
npm start
```

### Running Tests
```bash
cd server
npm test
```
